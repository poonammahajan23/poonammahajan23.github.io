# poonammahajan23.github.io

This repository is a static portfolio site for the GitHub Pages user site.
It will be served at: `https://poonammahajan23.github.io`

How to update locally:
1. Clone: `git clone https://github.com/poonammahajan23/poonammahajan23.github.io.git`
2. Add/edit files (index.html, assets/styles.css, assets/*)
3. Commit & push:
   ```
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```

Notes:
- For a user site (username.github.io) the repository name must match your username: `poonammahajan23.github.io`.
- GitHub Pages will serve the content from the repository root on the `main` branch by default. If your default branch differs, set the Pages source in Settings â†’ Pages.
- If you want a custom domain, add a `CNAME` file with the domain name in the repository and configure DNS accordingly.
